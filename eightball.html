The Magic 8 Ball method is meant to serve a sort of short-term fun program that provides a response to yes or no questions only. The user enters simply enters a yes or no question and the magic 8 ball is meant to provide a response to that question as seen in the first string array from the top. Words such as “is”, “were”, “would”, etc., that signify that a sentence would be a yes or no question are put into a text file and that text file and each item is checked if it matches the “starter” of the sentence that the user inputted. Furthermore, the sentence is checked to make sure it contains a question mark at the end since it should be in the form of a sentence. If the user did not provide a yes or no question, then the method would output a short print statement saying that their inputted statement is not valid, and will then prompt them again to input a yes or no question.

The purpose of the method for the ChatBot was simply to create a distinction between the two different types of questions that people could ask. We found that yes or no questions that certain starters that signify that the sentence would require a yes or no response and so the method relies on checking that the inputted sentence begins with that type of starter and thus prints a statement in response as if it were a magic 8 ball.
